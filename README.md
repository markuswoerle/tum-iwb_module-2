# Research Data
This repository contains the research data used in the manuscript "Time series analysis-based dimensioning of decentralized hydrogen production systems for manufacturing applications" by Markus Woerle, Erik Nitzschke, Daniel Schneider, and Michael F. Zaeh (under review, DOI: tbd.).

## Contact
Markus Woerle (markus.woerle@iwb.tum.de)

## Documentation
To replicate the results shown in the paper mentioned above, all the files that are stored in this repository are to be downloaded and to be saved in the same folder.

### MATLAB files
The following files contain the MATLAB code that was used to evaluate the case study in the paper:
- main.m
- periodicity.m
- dimensioning.m
- costs.m
- constraints.m
- storage.m
- soc.m

### Input file
The following file contains the input data of the case study described in the paper:
- hydrogen_demand_profile.csv
